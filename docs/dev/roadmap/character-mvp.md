## 2. MVP Milestones

### M1: Core Character Stats
- Character class/model definition
- Basic attributes (HP, Max HP, Strength, Defense, Speed)
- Stat getters and setters
- Character initialization
- Stat persistence/serialization

### M2: Skills and Abilities
- Skill class definition
- Skill learning system
- MP/resource costs for skills
- Skill effect implementation (damage, healing, buffs)
- Skill restrictions (level requirements, prerequisites)
- Active skill list management

### M3: Equipment System
- Equipment slots (weapon, armor, accessory)
- Item class hierarchy (base item, equipment, consumable)
- Equipment stat bonuses
- Equip/unequip functionality
- Equipment restrictions (class-specific, level requirements)

### M4: Inventory Management
- Inventory class with item storage
- Add/remove items
- Item stacking
- Inventory capacity limits
- Item usage (consumables)
- Item sorting and filtering

### M5: Character Progression
- Experience point system
- Level-up mechanics
- Stat growth on level-up
- Skill acquisition on leveling
- Experience gain calculation
- Level cap implementation

## 3. Beyond MVP: Future Features

### Advanced Character Classes
- Multiple character classes (Warrior, Mage, Rogue, etc.)
- Class-specific abilities and stat growth
- Class advancement/promotion system
- Dual-class or multiclass mechanics
- Class skill trees

### Character Customization
- Character name and appearance
- Custom stat allocation
- Talent/perk systems
- Character backstory and traits
- Customizable growth paths

### Advanced Stats and Mechanics
- Secondary stats (Accuracy, Evasion, Critical Rate)
- Stat soft caps and diminishing returns
- Hidden stats and mechanics
- Stat modifiers from buffs/debuffs
- Temporary stat boosts

### Enhanced Equipment
- Equipment rarity tiers
- Equipment upgrades and enhancement
- Set bonuses for matching equipment
- Cursed/enchanted items
- Weapon durability system
- Equipment crafting

### Character Relationships
- Party member relationships and bonds
- Relationship-based stat bonuses
- Character-specific dialogue and events
- Support conversations
- Relationship skill unlocks

### Save and Profile System
- Multiple character save slots
- Character import/export
- Achievement tracking
- Play statistics
- Character presets and templates
