## 2. MVP Milestones

### M1: Core Battle Loop
- Turn management system (player turn, enemy turn)
- Action selection interface
- Basic attack action implementation
- Battle state tracking (ongoing, victory, defeat)
- Battle initialization and cleanup

### M2: Combat Actions
- Attack action with damage calculation
- Defend/Guard action (damage reduction)
- Skill/ability system integration
- Item usage during battle
- Action validation and execution

### M3: Enemy AI
- Basic enemy AI decision-making
- Enemy action selection logic
- Multiple enemy type definitions
- Enemy behavior patterns
- Difficulty balancing

### M4: Battle UI
- Text-based battle display
- Action menu rendering
- Battle log/message system
- HP/status visualization
- Victory/defeat screens

## 3. Beyond MVP: Future Features

### Party-Based Combat
- Multiple player characters in battle
- Party member switching
- Coordinated attacks and combos
- Party-wide status effects

### Advanced Battle Mechanics
- Status effects (poison, stun, buff, debuff)
- Elemental weaknesses and resistances
- Critical hits and miss chances
- Counter-attacks and reaction abilities
- Multi-target attacks

### Battle Environments
- Environmental effects on combat
- Terrain advantages/disadvantages
- Weather effects
- Interactive battlefield elements

### Strategic Features
- Turn order display and manipulation
- Skill cooldowns and resource management
- Combo system
- Battle formations
- Escape/flee mechanics

### Visual Enhancements
- Graphical battle animations
- Character sprites and enemy models
- Battle backgrounds
- Visual effects for skills and attacks
- Camera movements and transitions
